## Recommendations

1. **Flush portfolio state and start fresh.** Current portfolio contaminated with pre-fix positions. The $3,120 loss is sunk cost from the bugs. Reset capital to $10,000 and let only post-fix logic trade.

2. **Add rule-based classifier for price-threshold pairs.** Detect patterns like "X above $Y" where the same metric with different thresholds is implication, not mutual_exclusion. Would catch PLTR, BTC, FDV threshold pairs.

3. **Extend crypto time-interval rule to match "above $X on date" patterns.** Current regex only matches "Up or Down ... TIME-TIME". Add a second regex for "above/below $X on DATE" patterns.

4. **Stop trading unconverged opportunities.** 65% of simulated opps didn't converge. Filter: only simulate if bregman_gap < 0.001. This would eliminate most noise trades.

5. **Raise min_edge further or add fee-aware filtering.** With 2% fees per leg, round-trip costs ~4% of notional. The 55% of opps with <2-cent edges are guaranteed losers.
